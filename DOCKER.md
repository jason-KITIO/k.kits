```powershell
.\docker.ps1 up      # Démarrer
.\docker.ps1 down    # Arrêter
.\docker.ps1 help    # Aide
```

### Linux/Mac
```bash
make up              # Démarrer
make down            # Arrêter
make help            # Aide
```
